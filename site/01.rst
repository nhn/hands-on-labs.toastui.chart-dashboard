****
개요
****

TOAST UI Chart는 FE개발랩에서 개발한 오픈소스 자바스크립트 라이브러리입니다. TOAST UI Chart를 이용하면 쉽고 아름답게 데이터를 이해할 수 있도록 도와줍니다. 이번 핸즈온랩스에서 TOAST UI Chart를 경험해보고 서비스에 적용해보세요!


학습할 내용
============

* TOAST UI Chart의 기능을 활용하여 대시보드를 만드는 방법

준비물
======

* 익숙한 편집기
* node.js
* 브라우저
