5. 도넛 차트에 추가 옵션 적용해보기¶

5.1. 도넛 차트의 showLabel옵션 적용방법 알아보기¶

  • https://github.com/nhnent/tui.chart/blob/production/docs/wiki/features-series.md#showing-label-on-series-area

5.2. 도넛 차트에 series.showLabel 옵션 적용하기¶

series.showLabel 옵션을 적용하여 도넛 차트 내에 수치를 표현합니다.

//...
const donutChartOptions = {
      //...
    series: {
        showLabel: true
    }
};

5.3. 도넛 차트에 chart.format 옵션 적용하기¶

series.showLabel 옵션을 적용하여 나타난 값이 number 포멧으로 나오지 않아서 어색한 부분을 확인할 수 있습니다. 앞서 살펴봤던 차트의 일반적인 기능을 소개하는 문서에서 값을 포멧팅하는 부분을 찾아 아래처럼 적용하여 숫자형식으로 포멧팅합니다.

const donutChartOptions = {
  chart: {
      ...
      format: '1,000'
  }
  ...
};

5.4. 도넛 차트에 series.showLegend 및 series.labelAlign 적용방법 알아보기¶

  • https://github.com/nhnent/tui.chart/blob/production/docs/wiki/chart-types-pie.md#displaying-a-legend-label-to-each-center-of-a-piece-in-the-pie-graph

5.5. 도넛 차트에 showLegend 옵션 적용하기¶

문서에서 확인한 옵션을 실제로 적용합니다.

//...
const donutChartOptions = {
    series: {
        showLegend: true,
        labelAlign: 'outer'
    }
};
